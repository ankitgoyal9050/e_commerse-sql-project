# e_commerse-sql-project

## 📌 Project Overview

This project demonstrates SQL skills using an E-Commerce dataset.
It covers real-world business scenarios such as sales analysis, customer behavior, and product performance.

The goal of this project is to practice and showcase SQL concepts required for a Data Analyst role.

---

## 🧰 Tools & Technologies Used

* SQL (PostgreSQL)
* Database Queries
* Data Analysis Concepts

---

## 📂 Dataset Description

The dataset represents an E-Commerce platform and includes:

* Customers Data
* Orders Data
* Products Data
* Sales Transactions
* order_items data
* returns data

---

## 🔑 SQL Concepts Covered

### 🔹 Aggregate Functions

* COUNT() → Count total records
* SUM() → Total sales calculation
* AVG() → Average values
* MAX() / MIN() → Highest & lowest values

---

### 🔹 String Functions

* UPPER() / LOWER() → Text formatting
* LENGTH() → Character count
* SUBSTRING() → Extract part of text
* CONCAT() → Combine columns

---

### 🔹 Numeric Functions

* ROUND() → Round values
* CEIL() / FLOOR() → Adjust numeric values
* ABS() → Absolute values

---

### 🔹 Date Functions

* NOW() → Current date & time
* CURRENT_DATE → Current date
* EXTRACT() → Get year/month

---

### 🔹 Conditional Functions

* CASE → Apply conditions (IF-ELSE logic)
* COALESCE() → Handle NULL values
* NULLIF() → Avoid errors

---

### 🔹 Window Functions (Advanced)

* ROW_NUMBER() → Unique row numbering
* RANK() → Ranking with gaps
* DENSE_RANK() → Ranking without gaps
* LAG() → Previous row value
* LEAD() → Next row value
